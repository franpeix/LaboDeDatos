Librerías utilizadas:

	duckdb
	pandas
	matplotlib
	numpy
	seaborn
	scikit-learn
	Pathlib


Instrucciones de cómo ejecutar el código:
	
	1- En la terminal, verificar que las librerías utilizadas se encuentran instaladas correctamente, ejecutar (sin comillas) : 'pip install duckdb, pandas, matplotlib, numpy, seaborn, scikit-learn, Pathlib'
	2- Verificar que la ruta actual en la que se está navegando contiene al directorio donde están los archivos
	3- Para ejecutar todo el código, colocar lo siguiente (sin comillas):  'python mnistc_seguidoresdenavathe.py'

	De igual forma, si se desea ejecutar algun apartado (ejercicio) en específico, se puede abrir el código desde spyder (colocar en la terminal, sin comillas 'spyder'), luego abrir el archivo 'mnistc_seguidoresdenavathe.py', colocar el path en donde se encuentra la fuente de datos mnistc (en nuestro caso el mismo se encontraba en 'Downloads' y ejecutar la celda. 
	Luego, ejecutar la sección correspondiente a los Imports, y por ultimo ejecutar la celda que se quiera observar.

	El archivo mnistc_seguidoresdenavathe.py implementa funciones que se encuentran en otros archivos. Si se desea ver la implementación de las funciones, deberán realizar los pasos mencionados arriba, pero en este caso con el nombre del archivo donde se encuentra el ejercicio (analisisExploratorio.py, clasificacionBinaria.py, clasificacionMulticlase.py)
